Button Components

Design system explorations — heritage modern direction. Five core variants covering the full action hierarchy.

Variant 01
Primary — Tobacco Solid
High-priority actions. "Add to Humidor", "Shop Now", "Checkout". One per screen.
Variant 02
Outlined — Heritage Stroke
Supporting actions alongside a primary. "Learn More", "Save", "Compare". Clean, deliberate, restrained.
Variant 03
Gold — Burnished Accent
Reserved for premium, membership, or aspirational moments. Use sparingly — this is the delight button.
Variant 04
Ghost — Quiet Action
Lowest hierarchy. "Cancel", "Skip", "View All →". Steps aside and lets primary actions breathe.
Variant 05
Destructive — Ember
Irreversible or high-risk actions. "Remove from Humidor", "Delete Review". Starts outlined, fills on hover to confirm intent.
All Variants
Loading States
Consistent spinner behavior across all variants using border-trick animation.
Context
On Dark Surface
Buttons on espresso-dark backgrounds — product detail screens, overlays, hero sections.
